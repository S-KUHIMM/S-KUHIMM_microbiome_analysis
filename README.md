# Ecological stratification of the human gut microbiome via S-KUHIMM

This repository contains the custom Python scripts (Jupyter Notebooks) and source data used for the statistical analyses and visualization in the manuscript:
**"Ecological stratification of the human gut microbiome resolves metabolic heterogeneity via precision substrate optimization"**

## Requirements
The code was executed in a Jupyter Notebook environment (Google Colaboratory) using the following core libraries:
- Python 3.x
- pandas
- numpy
- scipy
- matplotlib
- seaborn
- statsmodels

## Usage
All scripts (`.ipynb`) and source data (`.csv`) are included in this repository. 
To reproduce the figures and statistical results:
1. Download the files from this repository.
2. Ensure that the `.csv` data files are placed in the same directory as the `.ipynb` scripts.
3. Run the notebooks sequentially in a Jupyter environment (e.g., JupyterLab, Anaconda, or Google Colaboratory).
